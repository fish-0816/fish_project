Alien Invasion - 外星人入侵

一个基于 Python + Pygame 开发的 2D 射击小游戏。
玩家驾驶飞船对抗不断逼近的外星舰队，通过击落敌人获取分数、拾取道具并挑战更高关卡。当前版本已经加入了四向惯性移动、连击系统、三种道具、粒子爆炸效果以及程序生成音效等功能。

项目特色
四向移动：支持上下左右移动，带有速度、加速度和摩擦减速效果，手感更像“太空漂浮”。
外星舰队系统：外星人按网格生成，成批左右移动，触边后整体下移并反向。
多种射击模式：
普通射击
快速射击（提升子弹上限与射速）
三重射击（一次发射三颗散射子弹）
道具系统：
rapid：快速射击
shield：护盾，抵挡下一次伤害
multi：三重射击
连击系统：连续击杀可叠加倍率，获得更高分数。
粒子特效：击毁外星人时生成爆炸火花。
程序生成音频：无需外部音频文件，射击音效和背景音乐均由 numpy + pygame 在启动时合成。
暂停与重开：支持 P 暂停/继续，点击 Play 开始或重新开始游戏。
运行环境
Python 3.10+
pygame
numpy
安装依赖
pip install pygame numpy
运行方式

当前项目的主入口文件为 alien_invasion.py，直接运行即可启动游戏：

python alien_invasion.py

游戏启动后会先初始化设置、窗口、音效管理器、UI 元素、精灵组以及初始外星舰队，然后进入主循环。

操作方式
鼠标点击 Play：开始 / 重新开始游戏
← →：左右移动飞船
↑ ↓：上下移动飞船
SPACE：发射子弹
P：暂停 / 继续
Q：退出游戏
游戏规则
击毁一整波外星人后进入下一关
关卡越高，外星人和整体节奏会更快
外星人撞到飞船或到达屏幕底部都会造成伤害
玩家初始有 3 条生命，全部耗尽后游戏结束
道具由被击毁的外星人随机掉落
连续击杀会累积连击倍率，提高得分效率

.
├── alien_invasion.py   # 游戏入口与主循环

├── game_stats.py       # 运行时状态

├── ship.py             # 玩家飞船（惯性移动）
├── alien.py            # 外星人精灵
├── bullet.py           # 子弹精灵
├── powerup.py          # 道具胶囊
├── particle.py         # 爆炸粒子
├── button.py           # Play 按钮
├── sounds.py           # 程序生成音效与背景音乐
└── images/
    ├── ship.bmp
    └── alien.bmp

项目已将逻辑拆分为三个核心子模块：game_events.py、game_updates.py、game_ui.py，而 game_functions.py 作为统一 API 门面，使主循环不需要关心具体函数来自哪个子模块。

核心设计
1. 模块化结构

项目将输入、更新、渲染分离，便于后续扩展与维护：

game_events.py：处理所有输入事件
game_updates.py：处理物理、碰撞、生成、状态更新
game_ui.py：负责所有画面绘制与 HUD 显示
2. 惯性飞船移动

飞船不会立即达到最大速度，而是通过加速度逐渐加速，并在松键后通过摩擦减速，带来更顺滑的手感。

3. 程序化音频

本项目不依赖外部音频素材。射击音与背景音乐由代码实时合成，减少资源依赖，也让项目更有特色。
